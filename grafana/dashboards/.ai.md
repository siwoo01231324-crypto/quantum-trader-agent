# grafana/dashboards/

## 목적
Grafana 대시보드 JSON 정의 보관. `docker-compose.yml`로 띄운 Grafana가 자동 로드한다.

## 구조
- 파일명 형식: `{scope}.json` (system / strategy / execution)
- 각 대시보드는 4개 이상 패널, UTC 시간대 기본, Prometheus 데이터소스 참조

## 현재 파일 목록
- `system.json` — 시스템 수준 메트릭 (프로세스·큐 깊이·API 에러율·레이턴시) (#26)
- `strategy.json` — 전략 지표 (포지션·PnL·샤프·DD) (#26)
- `execution.json` — 주문/체결 지표 (체결률·슬리피지·거래량) (#26)

## 규칙
- 대시보드 편집 후 JSON Export로 반드시 재저장
- 민감 정보(URL 토큰 등) 포함 금지
- 신규 패널 추가 시 메트릭 네이밍 규약(`docs/specs/observability.md`) 준수
