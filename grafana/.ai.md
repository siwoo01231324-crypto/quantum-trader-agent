# grafana/

## 목적
Grafana 대시보드 JSON과 프로비저닝 설정.

## 구조
| 경로 | 역할 |
|------|------|
| `dashboards/system.json` | 프로세스/지연/킬스위치 등 시스템 지표 (5패널) |
| `dashboards/strategy.json` | 전략별 PnL/포지션/시그널/리스크 (4패널) |
| `dashboards/execution.json` | 발주↔체결, broker 실패율, 지연 히트맵 (4패널) |

## 규칙
- 모든 대시보드는 `uid`와 `tags` 포함
- 패널 4개 이상 (test_metrics.py로 검증)
