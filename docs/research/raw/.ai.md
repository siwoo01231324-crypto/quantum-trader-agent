# docs/research/raw/

## 목적
외부 자료의 **원본 전사·캡처** 를 그대로 저장한다. 가공·해석은 별도 background 노트에서 한다.

## 구조
파일별로 1개 자료. 파일명 패턴: `<주제>-<YYYY-MM-DD>.md`

## 작업 규칙
- `type: research` 프론트매터 필수 (`docs/schemas/note-schemas.md`)
- `sources` 필드에 원본 URL·채널·출연자·전사 도구 명시
- 본문 = 전사 원문 (mojibake·오타 그대로 유지하여 추적성 확보)
- 해석·요약·평가는 본문 상단의 별도 섹션으로 분리하여 원문과 구분
- 영상은 자동자막 (yt-dlp) 또는 Whisper 로 전사
- raw 자료를 인용하는 background 노트는 `[[<raw 파일 id>]]` wikilink 로 역참조

## 보안·저작권
- 비공개 영상·유료 강의 전사는 저작권 확인 후 저장 (내부 참고용)
- 출연자 PII (전화·이메일·주소) 는 자동 마스킹

## 관련 디렉토리
- `docs/background/` — 본 raw 자료에서 도출한 정제 조사 노트
- `docs/research/` — 부모 (raw 의 상위)
