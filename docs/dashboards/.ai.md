# docs/dashboards/

## 목적
Obsidian Dataview 플러그인으로 프론트매터를 집계·조회하는 대시보드 노트 모음. 볼트 사용자가 Obsidian 에서 열면 실시간으로 렌더링된다.

## 구조
```
docs/dashboards/
├── .ai.md                     ← 이 파일
├── strategies-live.md         ← 라이브 전략 리스트 + Sharpe
├── risk-coverage.md           ← 전략별 적용 리스크 규칙 매트릭스
├── recent-incidents.md        ← 최근 30일 incident
└── ml-model-usage.md          ← ML 모델을 사용하는 Signal 집계
```

## 사용법
- Obsidian 에서 `docs/` 를 볼트로 열고 Dataview 플러그인을 활성화한다.
- 각 대시보드는 `dataview` 코드블록을 포함한다. 뷰 모드 전환 시 표 렌더링.

## 참조
- 프론트매터 규약: [[note-schemas]]
- 온톨로지(SPARQL): `docs/ontology/queries/`
