# 자동매매 프로그램 구현을 위한 선행 리서치 + 구현 플랜 초안

## 목적
자동매매 프로그램을 만들기 위해 어떤 자료·기술·의사결정이 필요한지 **web-research로 먼저 싹 긁고**, 그 결과로 구현 접근 플랜 초안을 작성한다. 이후 후속 리서치 이슈들의 AC는 이 플랜에서 도출한다.

## 배경
- 도구·API부터 파고들기 전에 "무엇을 조사해야 하는지" 자체를 먼저 정리해야 한다.
- 본 이슈는 Phase 0·1 리서치 이슈들의 상위(메타) 이슈다.

## 완료 기준
- [ ] web-research로 자동매매 프로그램 구현에 필요한 주제·자료·결정 포인트 목록 도출 (최소 15개 이상)
- [ ] 구현 접근 플랜 초안 작성 (데이터 → 신호 → 주문 → 리스크 → 모니터링 흐름)
- [ ] 플랜에서 파생되는 "필요한 후속 리서치 목록" 정리 → 기존 Phase 0·1 이슈와 매핑

## 구현 플랜
1. web-research-specialist 에이전트로 2026년 기준 자동매매 구축 가이드/사례 수집
2. 수집 결과를 `docs/specs/research-plan.md` 에 정리
3. 플랜에서 도출된 추가 리서치 항목은 별도 백로그 이슈로 분리 제안
