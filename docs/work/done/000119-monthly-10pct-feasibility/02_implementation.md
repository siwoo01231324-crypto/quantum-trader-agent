### 3.7.1 Sleeve 정의

| **A — Conservative core** | (b) 보수: Sharpe ≥ 1.0, MDD halt -5%, L=1 | 카탈로그 5전략 등가중 |

| **C — Experimental** | (c) 게이트: Phase 1=Sleeve A 와 동일 → Phase 3 실측 Sharpe ≥ 1.5 시 Sleeve B 로 격상 | A/B 트랜지션 |

> Sleeve A 의 σ·MDD·Sharpe 는 분석적 합성. 정확값은 5전략 등가중 portfolio daily returns 시계열 측정 필요 (별도 PR).

### 3.7.3 비중 시나리오 통합 metrics (분석적 합성, ρ=0.0075 가정)

수식:
- 통합 평균: $\mu_P = \sum_i w_i \mu_i$
- 통합 변동성: $\sigma_P^2 = \sum_{i,j} w_i w_j \sigma_i \sigma_j \rho_{ij}$ (sleeve 간 상관 0.1 가정)
- 통합 MDD 상한: $|MDD_P| \leq \sum_i w_i |MDD_i|$ (동시 발생 가정의 보수 상한; 실제는 sleeve 격리 시 더 작음)

**Phase 1 (Sleeve C 보수 운영) — A:B:C 비중**:

| 비중 (A/B/C) | 통합 mu | 통합 σ (대략) | 통합 Sharpe | 통합 MDD 상한 | 비고 |
|--------------|---------|---------------|-------------|---------------|------|
| **70/20/10** | 14.46% | ~14% | ~1.03 | -25% | 보수 우세, MDD halt -25% 허용 시 안전 |
| 60/20/20 | 14.46% | ~14% | ~1.03 | -24% | C 격상 게이트 비중 확대 |
| 50/30/20 | 18.55% | ~18% | ~1.03 | -27% | 공격 비중 격상, MDD 약간 확대 |
| 90/10/0 | 10.37% | ~12% | ~0.86 | -20% | 보수 강화, 공격 제한 |

**Phase 3 (Sleeve C 격상 후) — A:B(통합) 비중 변화**:

| Phase 1 비중 | Phase 3 비중 (C → B) | 통합 mu | 통합 Sharpe | 통합 MDD 상한 |
|--------------|---------------------|---------|-------------|---------------|
| 70/20/10 | 70/30 | 18.55% | ~1.04 | -30% |
| 60/20/20 | 60/40 | 22.65% | ~1.05 | -35% |
| 50/30/20 | 50/50 | 26.74% | ~1.06 | -40% |

### 3.7.4 핵심 판정

- **단일 (a)/(b)/(c) 대비 sleeve 통합 Sharpe ~1.0 유지** — 보수 sleeve 의 안정성 + 공격 sleeve 의 알파 + 게이트 sleeve 의 데이터 의존 결정.
- **MDD halt 정책 변경 필요** — sleeve B 단독 -65% 이지만 portfolio 비중 0.2 적용 시 통합 MDD 상한은 -25% 수준으로 압축. 현 -5% halt 와 5배 차이 (단일 a 의 13배 대비 절반 수준).
- **운영 복잡도 추가** — sleeve 별 독립 Policy 인스턴스 필요. `risk-rule-dsl` v3 확장 (`sleeve_id` 필드).
- **자본 효율성** — KRX 1주 단위 + 거래세 0.20% (`[[20-position-sizing]]` §8) 에서 sleeve 가 너무 작으면 비중 이산화 오차 발생. 최소 sleeve 크기 = $10k 권장 (1주 가격 30만원 가정 시 슬리브당 30종목 보유 가능).
