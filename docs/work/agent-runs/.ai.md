# docs/work/agent-runs/

## 목적
`services/doc_agent` 가 초안 노트를 생성할 때마다 실행 메타(타임스탬프·생성기·출력 경로)를 JSON 한 건 기록하는 감사 로그 디렉토리.

## 구조
- `{YYYY-MM-DD-HHMMSS}-{generator}.log` — 실행 1건당 1 파일
  - `generator ∈ {backtest, incident, postmortem}`
  - 내용: `{"ts": ..., "generator": ..., "id": ..., "path": ...}`

## 규칙
- 사람이 손으로 수정하지 않는다 (자동 생성 전용).
- 볼륨이 커지면 `.gitignore` 에 `docs/work/agent-runs/*.log` 추가를 고려 (현 단계는 커밋 허용).
- 로그 작성 실패는 초안 생성기의 성공에 영향을 주지 않는다 (best-effort).

## 연관
- 생성기: `services/doc_agent/generators.py`
- 서브에이전트: `.claude/agents/doc-writer.md`
