# docs/whitepaper/

## 목적
프로젝트의 기술 백서 및 심층 기술 문서를 보관한다. 아키텍처 설계 근거, 기술 선택 이유, 알고리즘 설명 등 프로젝트의 핵심 기술을 깊이 있게 기술한다.

## 구조
- 파일명 형식: `{주제}.md` (예: `ai-engine-architecture.md`)
- 각 문서에는 배경·목표·기술 상세·결론을 포함한다

## 규칙
- 기술적 주장에는 근거(벤치마크, 논문, 실험 결과 등)를 첨부한다
- 다이어그램이 필요한 경우 Mermaid 문법을 사용한다
- 문서 하단에 참고 문헌을 명시한다
