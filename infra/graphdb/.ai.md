# infra/graphdb/

## 목적
Ontotext GraphDB Free 10.6.4 를 Docker Compose 로 기동하고, `qta` 리포지토리 초기 설정(RDF config TTL)을 보관한다.

## 구조
```
infra/graphdb/
├── .ai.md             # 이 파일
├── docker-compose.yml # GraphDB 서비스 정의 (포트 7200, healthcheck)
├── repo-config.ttl    # GraphDB FreeSailRepository 설정 (qta 리포)
├── data/              # 런타임 트리플 스토어 데이터 (gitignore)
└── logs/              # 런타임 로그 (gitignore)
```

## 역할
- `docker-compose.yml` — `GDB_PORT`(기본 7200) / `GDB_JAVA_OPTS`(기본 -Xmx2g -Xms512m) 환경변수 오버라이드 지원
- `repo-config.ttl` — `scripts/graphdb_bootstrap.py` 가 리포 생성 시 POST로 업로드
- T-Box + A-Box 는 모두 **default graph** 에 저장 (기존 `.rq` 쿼리 파일 무수정 호환)
- 부트스트랩 절차: `docker compose up -d` → `python scripts/graphdb_bootstrap.py`

## 운영 참고
- 포트 충돌 시: `GDB_PORT=7201 docker compose up -d`
- JVM 메모리 조정: `GDB_JAVA_OPTS="-Xmx1g -Xms256m" docker compose up -d`
- 런북: `docs/runbooks/graphdb-ops.md`
