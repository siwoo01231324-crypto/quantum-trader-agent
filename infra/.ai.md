# infra/

## 목적
로컬 개발 및 CI 환경에서 사용하는 인프라 서비스(GraphDB 등)의 Docker Compose 설정과 초기화 파일을 관리한다.

## 구조
```
infra/
└── graphdb/          # Ontotext GraphDB Free 10.6.4 서비스 설정
    ├── .ai.md
    ├── docker-compose.yml
    ├── repo-config.ttl
    ├── data/         # 런타임 데이터 (gitignore)
    └── logs/         # 런타임 로그 (gitignore)
```

## 역할
- `graphdb/` — GraphDB 컨테이너 기동, 리포지토리 초기화 설정 보관
- 각 하위 디렉토리는 독립적인 서비스 단위로 관리
- 런타임 생성 데이터(`data/`, `logs/`)는 `.gitignore`로 제외
