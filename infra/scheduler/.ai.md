# infra/scheduler/

## 목적

Windows Task Scheduler XML 정의 파일 — 데몬·routine 의 OS-level 자동 시작/주기 실행 설정.

## 구조

- `shadow_swing_task.xml` — KIS swing strategy shadow live daemon Task Scheduler 등록 정의 (#143 R4/R6 계열).
- `shadow_swing_r6_task.xml` — 동일, R6 변형.

## 운영 규칙

- `scripts/setup_airborne_alert_task.ps1` 와 유사하게 `schtasks /create /xml <path>` 로 등록.
- XML 안에 절대 경로가 박혀 있을 수 있어 호스트별 수정 후 등록 (사용자 PC 환경 가정).
- 새 Task XML 을 만들 때 본 디렉토리에 추가, 1줄 설명을 위 "구조" 섹션에 추가.

## 관련

- [[project_30day_daemon_hosting]] — 30일 운영 호스팅 메모 (로컬 PC 가동 확정).
- `scripts/setup_airborne_alert_task.ps1` — airborne 데몬용 등록 스크립트 (#261).
