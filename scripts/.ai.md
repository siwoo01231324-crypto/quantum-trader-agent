# scripts/

## 목적
CI 및 로컬에서 실행되는 유틸리티 스크립트를 보관한다.

## 파일
| 파일 | 역할 |
|------|------|
| `check_invariants.py` | 아키텍처 불변식 검증 (위반 시 CI 차단, `--strict` 기본 CI) |
| `check_forbidden_files.py` | 금지 파일 커밋 여부 검사 |
| `ontology_sync.py` | 프론트매터 → `docs/ontology/instances.ttl` 동기화 (#47) |
| `migrate_frontmatter.py` | `docs/**/*.md` 프론트매터 일괄 주입 (경로 기반 type 추론, idempotent) (#52) |
| `shacl_validate.py` | SHACL 제약 기반 도메인 규칙 검증 (pyshacl, `--strict` 시 fail) (#54) |

## 사용법
```bash
python scripts/check_invariants.py --strict
python scripts/check_forbidden_files.py
python scripts/migrate_frontmatter.py --dry-run
python scripts/migrate_frontmatter.py --apply
python scripts/ontology_sync.py --check
python scripts/shacl_validate.py --strict
```

## 규칙
- 새 불변식 추가 시 `check_invariants.py`에 검사 함수를 작성하고 `CLAUDE.md`에도 명시
