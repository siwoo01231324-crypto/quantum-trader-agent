## Shadow Live Loop (#80)

- `shadow_run.py` — Phase 1 Shadow Live Loop CLI 진입점
  - 사용법: `python scripts/shadow_run.py --symbols BTCUSDT,ETHUSDT --duration 8h`
  - BinancePublicFeed (aggTrade WS, 키 불요) 자동 연결
  - production.yaml 미존재 시 fallback (Phase 1 stub, #94 머지 후 활성화)
  - WAL/Lock: `logs/shadow/{run_id}/`
- `shadow_report.py` — WAL 파싱 + 일별 PnL → daily return 시계열 → `register_strategy_returns()` 호출
  - Sharpe 비교 4조건 강제 (동일 데이터 소스/슬리피지/수수료/사이징)
  - 4조건 불일치 시 fail 처리

### 운영 흐름

1. shadow_run.py 로 매일 장중 시간 실행 (10거래일 누적)
2. shadow_report.py 로 daily return + Sharpe 비교 → `02_implementation.md` 표 생성
3. Exit Criteria 5종 자동 검증

## 사용법
```bash
python scripts/fetch_candles.py --symbol BTCUSDT --interval 15m
python scripts/run_backtest.py --strategy momo-btc-v2 --data-dir lake/
python scripts/compare_momo_btc_v2_sizing.py --data-dir lake/
python scripts/train_metalabeler_btc.py --output-dir models/momo-btc-v2
python scripts/retrain_metalabeler.py --strategy-id momo-btc-v2 --lake-dir lake/ --bench
python scripts/shadow_run.py --symbols BTCUSDT,ETHUSDT --duration 8h
python scripts/shadow_run.py --symbols BTCUSDT --max-iterations 100  # 테스트용
python scripts/check_invariants.py --strict
python scripts/check_forbidden_files.py
python scripts/migrate_frontmatter.py --dry-run
python scripts/migrate_frontmatter.py --apply
python scripts/ontology_sync.py --check
python scripts/ontology_sync.py --push-graphdb   # GraphDB 로 A-Box 업로드 (#55)
python scripts/shacl_validate.py --strict
python scripts/graphdb_bootstrap.py              # 로컬 GraphDB 초기화 (#55)
```
