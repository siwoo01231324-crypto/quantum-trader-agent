# scripts/

## 목적
CI 및 로컬에서 실행되는 유틸리티 스크립트를 보관한다.

## 파일
| 파일 | 역할 |
|------|------|
| `check_invariants.py` | 아키텍처 불변식 검증 (위반 시 CI 차단, `--strict` 기본 CI) |
| `check_forbidden_files.py` | 금지 파일 커밋 여부 검사 |
| `ontology_sync.py` | 프론트매터 → `docs/ontology/instances.ttl` 동기화 (#47) |
| `migrate_frontmatter.py` | `docs/**/*.md` 프론트매터 일괄 주입 (경로 기반 type 추론, idempotent) (#52) |
| `shacl_validate.py` | SHACL 제약 기반 도메인 규칙 검증 (pyshacl, `--strict` 시 fail) (#54) |
| `graphdb_client.py` | GraphDB REST 공용 HTTP 헬퍼 (wait_for_ready, create_repo, upload_ttl, sparql_update) (#55) |
| `graphdb_bootstrap.py` | GraphDB `qta` 리포지토리 생성 + T-Box (`trading.ttl`) 로드 (#55) |

| `fetch_candles.py` | Binance REST API → OHLCV Parquet 데이터 수집 CLI (#67) |
| `run_backtest.py` | 백테스트 전략 실행 CLI — Sharpe/MDD 출력, 프론트매터 업데이트, doc_agent 초안 생성 (#67) |
| `compare_momo_btc_v2_sizing.py` | momo-btc-v2 `full` · `half-kelly` · `vol-target` 3 모드 동시 백테스트 → Sharpe·MDD·total_return·trades JSON 저장 (#69) |

## 사용법
```bash
python scripts/fetch_candles.py --symbol BTCUSDT --interval 15m
python scripts/run_backtest.py --strategy momo-btc-v2 --data-dir lake/
python scripts/compare_momo_btc_v2_sizing.py --data-dir lake/
python scripts/check_invariants.py --strict
python scripts/check_forbidden_files.py
python scripts/migrate_frontmatter.py --dry-run
python scripts/migrate_frontmatter.py --apply
python scripts/ontology_sync.py --check
python scripts/ontology_sync.py --push-graphdb   # GraphDB 로 A-Box 업로드 (#55)
python scripts/shacl_validate.py --strict
python scripts/graphdb_bootstrap.py              # 로컬 GraphDB 초기화 (#55)
```

## 규칙
- 새 불변식 추가 시 `check_invariants.py`에 검사 함수를 작성하고 `CLAUDE.md`에도 명시
