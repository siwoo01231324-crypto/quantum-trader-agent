# quantum-trader-agent (루트)

## 목적
한국 개인투자자용 **저빈도(LFT) 규칙기반·퀀트 팩터 자동매매 에이전트**. 브랜딩상 "quantum"을 차용하나 양자컴퓨팅과 무관(#5). Claude Code + oh-my-claudecode 기반 워크플로우로 이슈→PR→머지를 자동화.

## 구조
| 경로 | 역할 |
|------|------|
| `CLAUDE.md` | 아키텍처 불변식·규칙·작업 흐름 (세션 시작 시 필독) |
| `AGENTS.md` | 레포 전체 목차·구조 트리 |
| `.github/workflows/` | CI (project 자동화 · ontology-check · mcp-smoke · build-exe #123) |
| `.claude/` | 커맨드·서브에이전트(doc-writer)·훅 정의 |
| `docs/` | **Obsidian 볼트** (SOT) — specs/schemas/ontology/dashboards/onboarding/runbooks/work/… |
| `src/` | 애플리케이션 코드 (data_lake/risk/execution/observability/ops/tax) |
| `services/` | LLM 연동 레이어 (obsidian_mcp #51 · doc_agent #53) |
| `scripts/` | check_invariants · ontology_sync · migrate_frontmatter |
| `tests/` | pytest 스위트 + fixtures (샘플 볼트 · doc_agent 입력) |
| `policies/` | 리스크 정책 YAML (#24) |

| `setup.sh` | 템플릿 초기화 (1회) |
