# src/ml/regime/ — HMM 기반 시장 체제 판별

## 목적
Hamilton (1989) 2/3-state Gaussian HMM과 threshold-based classifier로 시장 regime 판별.
PR #172 의 PBO 미통과를 regime-conditional strategy switching으로 공략하기 위한 핵심 모듈.

## 파일 구성
| 파일 | 역할 |
|------|------|
| `__init__.py` | 서브패키지 re-export (`GaussianHMMRegime`, `RegimeResult`, `ThresholdRegime`) |
| `hmm.py` | Gaussian HMM regime detector — Baum-Welch EM + Viterbi decoding |
| `threshold.py` | Rule-based threshold classifier — rolling return + funding rate |

## 공개 API
```python
from src.ml.regime import GaussianHMMRegime, RegimeResult, ThresholdRegime
```

## 외부 의존성
- `hmmlearn>=0.3,<1` (pyproject.toml main deps)
- `numpy>=1.26`
- `pandas>=2.1`

## 호출 경로
`regime_switching.py` (swing bench) → `GaussianHMMRegime.fit_predict()` → `RegimeResult`
→ state 기반으로 S2c/S4 전략 선택

## 불변식
- `random_state=42` — 재현성 강제
- NaN 자동 제거 (fit/predict 전)
- State 라벨링은 variance 기준 (low_vol/high_vol) — EM label switching 방지
- LLM 개입 금지 — 순수 통계 모델

## 볼트 사용 안 함
순수 수치 계산 모듈. Obsidian MCP 서버 연결 불필요.
