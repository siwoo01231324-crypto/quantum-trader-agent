# src/

## 목적
프로덕션 코드 루트. 도메인별 서브패키지로 분리된다.

## 구조
| 경로 | 역할 |
|------|------|
| `execution/` | 주문 실행 알고리즘 (TWAP/VWAP/Limit/Market) + KRX 단일가 핸들러 (#25) |
| `observability/` | Prometheus 메트릭 카탈로그 (#26) |
| `ops/` | Kill Switch·자동 트리거 3종·수동 CLI (#27) |
| `tax/` | KR 개인 세법 자동화 (FIFO 매칭·양도세/거래세/배당세·CSV 리포터) (#28) |

## 규칙
- 외부 I/O는 어댑터 계층에서만 (도메인 모듈은 순수)
- 새 패키지 추가 시 본 파일 표 갱신
