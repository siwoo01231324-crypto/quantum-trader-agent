# src/signals/

## 목적
무상태(stateless) 시그널 계산 함수. 입력 Series → 출력 Series.

## 구조
- `rsi.py` — RSI 계산 (Wilder smoothing) + divergence 감지 (rolling min/max)

## 규칙
- 모든 시그널은 `shift(1)` (lag-1) 사용 — 룩어헤드 바이어스 방지 필수
- 순수 함수: 외부 상태 의존 없음, 동일 입력 → 동일 출력
