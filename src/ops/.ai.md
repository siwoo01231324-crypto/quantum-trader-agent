# src/ops — 운영(Ops) 레이어

## 목적
Kill switch / DR 게이트와 자동 트리거, 수동 CLI 를 제공한다.

## 구조
- `kill_switch.py` — `KillSwitch` 게이트 (`allow_order` / `trip` / `release`)
- `triggers.py` — `DrawdownTrigger`, `ApiErrorTrigger`, `FillAnomalyTrigger`
- `cli.py` — `python -m src.ops.cli {kill|release|status}`

## 규칙
1. 모든 신규 주문은 `KillSwitch.assert_allow_order()` 또는 `allow_order()` 통과 필수.
2. 청산 주문은 `liquidation=True` 로 게이트 통과(화이트리스트).
3. 자동 트리거는 `KillSwitch.trip(source="auto:*")` 로 통일된 경로를 사용.
4. 운영 상태 영속화는 `KILL_STATE_PATH` 환경변수 또는 기본 `.ops/kill_state.json`.
