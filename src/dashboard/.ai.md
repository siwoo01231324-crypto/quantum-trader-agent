# src/dashboard/

## 목적
FastAPI 로컬 대시보드 — localhost:8000 에서 4사분면 HTML UI 와 Prometheus `/metrics` endpoint 제공.

## 구조
| 파일 | 역할 |
|------|------|
| `app.py` | FastAPI 앱 팩토리 `create_app(state)` + standalone `app` 인스턴스 |
| `__init__.py` | 패키지 |

## 4사분면 레이아웃
| 사분면 | 내용 |
|--------|------|
| Q1 (좌상) | 손익 — 실시간·일간·월간 PnL 수치 |
| Q2 (우상) | 6종 한도 사용률 게이지 (per_trade·per_day·per_portfolio·per_position·sector·drawdown) |
| Q3 (좌하) | 신호→메타라벨러→주문→체결 타임라인 (최근 50건) |
| Q4 (우하) | 비상정지 4 트리거 상태 + 마지막 발동 시각 + 수동 발동/해제 버튼 |

## Endpoints
| Method | Path | 설명 |
|--------|------|------|
| GET | `/` | 4사분면 HTML (5초 자동 새로고침) |
| GET | `/metrics` | Prometheus exposition format |
| GET | `/api/pnl` | 손익 JSON |
| GET | `/api/limits` | 한도 사용률 JSON |
| GET | `/api/kill-switch` | 킬스위치 상태 JSON |
| POST | `/api/kill-switch/trigger` | 킬스위치 트리거 발동 |
| POST | `/api/kill-switch/reset` | 킬스위치 트리거 해제 |

## 실행
```bash
uvicorn src.dashboard.app:app --host 0.0.0.0 --port 8000 --reload
```

## 의존성
- `src/observability/metrics.py` — Prometheus `Metrics` 객체 (registry 공유)
- `fastapi`, `prometheus-client`, `uvicorn`

## 규칙
- `DashboardState` 를 통해 런타임 데이터 주입 — 직접 전역 상태 수정 금지
- `/metrics` 는 `src/observability/metrics.py` 의 registry 그대로 노출 (이중 등록 금지)
- LLM 위임 금지 (CLAUDE.md 불변식 #6)
