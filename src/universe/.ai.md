# src/universe/

## 목적
유니버스 필터링·랭킹·등급화 순수 함수 모듈. 거래 대상 종목 풀 정의 + 자산 안정성 평가.

## 구조
| 파일 | 역할 |
|------|------|
| `stability_grade.py` | `StabilityGrade.grade()` — 시가총액·거래량·개발활동 기반 A~F 등급 산출 순수 함수 |
| `kospi200.py` | KOSPI200 구성종목 정적 리스트 (pin-date 2026-04-25). `KOSPI200_CONSTITUENTS: list[dict]` (code/name/sector), `get_codes() -> list[str]`. 197개 종목. |
| `krx_calendar.py` | KRX 거래일/휴장일 판정 헬퍼. `is_krx_holiday(date) -> bool`, `is_krx_trading_hours(ts: datetime) -> bool` (KST 09:00~15:30, 평일, 비휴일), `next_session_open(now: datetime) -> datetime` (다음 KRX 영업일 09:00 KST, #216 — `src/live/schedule.py:wait_until_session_open` 가 사용). 2025-2026 정적 휴장일 내장. |
| `krx_pool.py` | KOSPI200에서 섹터 라운드로빈 샘플링. `get_pool_codes(n=30, *, sectors=None, seed=42) -> list[str]`, `get_pool(n=30, ...) -> list[dict]`. 005930(삼성전자) 항상 포함. 동일 seed 결정론적. |
| `krx_top.py` | KRX 시총 top-N universe builder (#218). `top_n_by_marcap(snapshot_df, market, n)`, `combined_top_n(snapshot_df, kospi_n, kosdaq_n)`, `filter_by_min_marcap(...)`. 호출자가 시총 스냅샷 DataFrame 공급. |
| `binance_top.py` | Binance USDT spot 24h 거래량 top-N universe builder (#218). `top_n_by_volume(snapshot_df, n, *, excluded_bases, excluded_suffixes)`. 스테이블/페그/레버리지 자동 제외. 호출자가 24h 스냅샷 공급. |
| `__init__.py` | `StabilityGrade`, `grade_symbol`, `kospi200`, `krx_calendar` 공개 API |

## 규칙
- 모든 함수는 **순수 함수(pure function)** — 외부 네트워크/DB/파일 I/O 없음.
- CoinGecko 등 외부 데이터 어댑터는 별도 이슈에서 구현 (follow-up chore).
- DSL 배선(`PerPortfolioRisk.stability_grade_min` + `evaluate_order` 필터)도 별도 이슈.
- `dev_activity=None` 시 가중치 자동 재분배: mcap 0.5 / volume 0.5.
- 특허 회피: 업리치 특허 청구항 (d) 등급 개념 참조, 입력 변수·기준 재정의.
