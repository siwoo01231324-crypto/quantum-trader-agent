# src/universe/

## 목적
알트코인 유니버스 필터링·등급화 모듈. 거래 대상 자산의 안정성을 정량적으로 평가한다.

## 구조
| 파일 | 역할 |
|------|------|
| `stability_grade.py` | `StabilityGrade.grade()` — 시가총액·거래량·개발활동 기반 A~F 등급 산출 순수 함수 |
| `__init__.py` | `StabilityGrade`, `grade_symbol` 공개 API |

## 규칙
- 모든 함수는 **순수 함수(pure function)** — 외부 네트워크/DB/파일 I/O 없음.
- CoinGecko 등 외부 데이터 어댑터는 별도 이슈에서 구현 (follow-up chore).
- DSL 배선(`PerPortfolioRisk.stability_grade_min` + `evaluate_order` 필터)도 별도 이슈.
- `dev_activity=None` 시 가중치 자동 재분배: mcap 0.5 / volume 0.5.
- 특허 회피: 업리치 특허 청구항 (d) 등급 개념 참조, 입력 변수·기준 재정의.
