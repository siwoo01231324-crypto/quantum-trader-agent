# src/brokers/

## 목적
KIS(한국투자증권) / Binance Futures 브로커 어댑터. 주문·체결·포지션·잔고 인터페이스 통일.

## 구조
- `types.py` — BrokerFill (Decimal 기반, C1 핵심)
- `base.py` — BrokerAdapter Protocol + 공통 자료형 (OrderRequest, OrderAck, Position, Balance)
- `errors.py` — BrokerError 계층
- `is_estimator.py` — Implementation Shortfall 사전 추정(`pre_flight_is_estimate`) + 사후 실현 IS(`realized_is`) + `MarketSnapshot` (#114)
- `rate_limiter.py` — named bucket 다중 모델
- `client_id.py` — Binance용 client_order_id 생성 (regex 검증)
- `logging_filter.py` — SecretMaskingFilter
- `config.py` — env loader (누락 시 명확한 오류)
- `router.py` — OrderRouter + ExecutionCostEstimator + active_broker 스왑 안전 프로토콜
- `protective_orders.py` (#127) — `ProtectiveOrderManager`: 진입 시 stop-loss + take-profit 자동 등록, 청산 시 자동 취소, PC 재기동 시 sync. Binance Futures `STOP_MARKET`/`TAKE_PROFIT_MARKET` (reduceOnly + MARK_PRICE) / KIS 조건부 주문 (예약매도, v2 통합 예정)
- `binance/` — Binance USDS-M Futures testnet 어댑터
- `kis/` — 한국투자증권 Open API paper 어댑터 (주 모듈)

## 규칙
- **Decimal 전용** (float 금지) — 가격·수량 모두 Decimal
- 시크릿은 env 전용, 하드코딩 절대 금지
- 기본 `paper=True` — 실전 전환은 명시적
- LLM 호출 금지 (CLAUDE.md 불변식)
- `place_order` 진입 시 `KillSwitch.assert_allow_order` 게이트 필수
- **sync RateLimiter 와 AsyncTokenBucket 을 한 파일에 혼재 절대 금지**:
  - `rate_limiter.py` — sync 전용, "초과 시 즉시 `RateLimitError` raise" 시맨틱
  - `async_rate_limiter.py` — async 전용, "초과 시 `await` 으로 token 대기 후 acquire" 시맨틱
  - 두 파일의 시맨틱이 근본적으로 다르므로 절대 합치지 말 것

## Async 레이어 (issue #73)
- `base.py` — `AsyncBrokerAdapter` Protocol 추가 (sync `BrokerAdapter` 와 병존, 별도 파일 금지)
- `errors.py` — `BrokerClosedError`, `ListenKeyExpiredError`, `WSDisconnectedError` 추가
- `async_rate_limiter.py` — `AsyncTokenBucket` (await-wait 시맨틱) + `AsyncBinanceRateLimiter`
- `async_backoff.py` — `exponential_backoff(attempt, base, cap, jitter)` + `backoff_sequence` async 제너레이터
- `binance/async_adapter.py` — Binance `AsyncBrokerAdapter` 구현 (httpx + websockets)
- `binance/async_http.py` — `httpx.AsyncClient` signed REST + `AsyncTokenBucket`
- `binance/async_ws.py` — user data stream + market stream (websockets>=12)
- `binance/listen_key.py` — listenKey 발급/keepalive/삭제
- `kis/async_adapter.py` — KIS `AsyncBrokerAdapter` 구현
- `kis/async_http.py` — KIS REST async
- `kis/async_ws.py` — KIS 체결 통보 WS
- async 함수 내 `time.sleep` / `requests` / `threading.Lock` / `httpx.Client`(sync) / `asyncio.run()` 절대 금지
