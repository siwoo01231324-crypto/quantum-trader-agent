# src/brokers/

## 목적
KIS(한국투자증권) / Binance Futures 브로커 어댑터. 주문·체결·포지션·잔고 인터페이스 통일.

## 구조
- `types.py` — BrokerFill (Decimal 기반, C1 핵심)
- `base.py` — BrokerAdapter Protocol + 공통 자료형 (OrderRequest, OrderAck, Position, Balance)
- `errors.py` — BrokerError 계층
- `rate_limiter.py` — named bucket 다중 모델
- `client_id.py` — Binance용 client_order_id 생성 (regex 검증)
- `logging_filter.py` — SecretMaskingFilter
- `config.py` — env loader (누락 시 명확한 오류)
- `router.py` — OrderRouter + active_broker 스왑 안전 프로토콜
- `binance/` — Binance USDS-M Futures testnet 어댑터
- `kis/` — 한국투자증권 Open API paper 어댑터 (주 모듈)

## 규칙
- **Decimal 전용** (float 금지) — 가격·수량 모두 Decimal
- 시크릿은 env 전용, 하드코딩 절대 금지
- 기본 `paper=True` — 실전 전환은 명시적
- LLM 호출 금지 (CLAUDE.md 불변식)
- sync 전용 (async는 후행 이슈 #73)
- `place_order` 진입 시 `KillSwitch.assert_allow_order` 게이트 필수
