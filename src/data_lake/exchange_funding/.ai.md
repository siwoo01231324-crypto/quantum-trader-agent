# src/data_lake/exchange_funding/

## 목적
이슈 #174 — 다중 거래소 (OKX, Bybit) funding rate fetcher. Binance 기존 패턴 확장.

## 구조
| 파일 | 역할 |
|------|------|
| `__init__.py` | `FundingFetcher` Protocol 정의 (공통 인터페이스) |
| `okx.py` | OKX `/api/v5/public/funding-rate-history` fetcher. cursor 기반 역방향 pagination (100건/req). |
| `bybit.py` | Bybit `/v5/market/funding/history` fetcher. startTime 전진 pagination (200건/req). |

## 공통 인터페이스
```python
fetch_funding_history(symbol: str, start: str, end: str) -> pd.DataFrame
# columns: [ts (UTC DatetimeTZ), funding_rate (float64)]
```

## Parquet 파티션 경로
```
lake/funding_rate/exchange=okx/symbol=BTC-USDT-SWAP/part-0.parquet
lake/funding_rate/exchange=bybit/symbol=BTCUSDT/part-0.parquet
```
