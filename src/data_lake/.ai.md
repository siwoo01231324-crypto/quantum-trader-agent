# src/data_lake/

## 목적
프로젝트 데이터 레이크의 스키마 정의와 기본 검증 헬퍼.

## 구조
- `schema.py` — 모든 테이블(OHLCV/Orderbook/Trade/Factor/Meta) 스키마와 `validate_schema`, `partition_path` 헬퍼.
- `__init__.py` — 공개 API 재노출.

## 역할
- 적재/조회 코드는 반드시 여기 정의된 스키마를 import해서 사용.
- 컬럼 추가 시 `docs/specs/data-lake-schema.md` 의 운영 규칙(섹션 7) 준수.
