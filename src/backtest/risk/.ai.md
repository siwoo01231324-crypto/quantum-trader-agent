# src/backtest/risk/

## 목적
단일 trade 의 intra-bar stop-loss / take-profit 시뮬레이터.  
이슈 #147 (VWMA + stop/take 통합 backtest) 의 핵심 모듈.

## 파일 구성
| 파일 | 역할 |
|------|------|
| `stop_take.py` | `simulate_stop_take()` — intra-bar 가격 경로 기반 stop/take 판정 |
| `__init__.py` | 패키지 공개 API |

## 설계 결정
- **intra-bar conservative 가정**: 같은 bar 에서 stop 과 take 가 모두 hit 가능한 경우 stop 우선 (불리한 쪽 채택 — 과최적화 방지).
- **gap-down/gap-up 처리**: bar open 이 entry 보다 이미 stop 아래일 경우 open 가격으로 exit (슬리피지 없이 gap 전체를 손실로 기록).
- **slippage 가정**: stop/take hit 시 0.05% (5 bps) slippage 추가 (`SLIPPAGE_PCT = 0.0005`). signal_exit 시에는 슬리피지 없음 (엔진이 별도 적용).
- **결과 필드**: `triggered_at` (bar timestamp or None), `exit_price`, `reason` ("stop" | "take" | "signal_exit" | None).

## 의존
- numpy, pandas 표준 라이브러리만 사용 (외부 의존 없음)
