# src/backtest/iranyi/ — Iranyi 12룰 D0~D9 Variant 라우터 (이슈 #185)

## 목적
Iranyi 12-rules 기반 D0~D9 frozen variant matrix 의 진입 신호 합성 패키지.
`VARIANT_REGISTRY` 가 SHA-256 으로 동결되어 있으며, 사후 변경 시 새 이슈 필요.

## 파일 구성
| 파일 | 역할 |
|------|------|
| `__init__.py` | 패키지 초기화 |
| `entry_router.py` | `VARIANT_REGISTRY` (D0~D9 frozen) + `variant_registry_sha256()` + `route(variant_id, bar, history, context)` |

## 공개 API
```python
from src.backtest.iranyi.entry_router import (
    VARIANT_REGISTRY,
    variant_registry_sha256,
    route,
)
```
