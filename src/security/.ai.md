# src/security

Windows DPAPI 기반 API 키 저장소.

## 목적
API 키 평문 노출 차단. Windows CryptProtectData/CryptUnprotectData 로 사용자 SID 바인딩 암호화.
다른 사용자 계정에서는 복호화 불가 (OS 강제).

## 구조
- `dpapi.py` — CryptProtectData/CryptUnprotectData 래퍼 (Windows-only)
- `key_store.py` — KeyStore 클래스: put/get/rotate/delete/list_keys
- `rotate_cli.py` — CLI 도구: `python -m src.security.rotate_cli`
- `__init__.py` — 공개 인터페이스

## 저장 경로
`%APPDATA%/qta/secrets/<KEY_NAME>.blob` — DPAPI 암호화 바이너리
`%APPDATA%/qta/secrets/<KEY_NAME>.meta` — 메타데이터 JSON (이름, 갱신 시각; 비밀값 없음)

## 평문 노출 방지 원칙
- .blob 파일: DPAPI 암호문만 저장
- .meta 파일: 이름과 타임스탬프만 저장, 비밀값 없음
- 로그/예외 메시지에 비밀값 출력 금지
- 환경변수 경유 없음

## 플랫폼
Windows 전용. 비-Windows 에서 import 시 RuntimeError 발생.
Linux/Mac 지원 필요 시 `keyring` 라이브러리로 대체 가능 (현재 미구현).
