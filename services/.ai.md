# services/

## 목적
LLM·외부 에이전트 연동 레이어. 트레이딩 핵심 로직(`src/`)과 분리해 보조 기능만 여기에 둔다.

## 구조
| 디렉토리 | 역할 | 관련 이슈 |
|----------|------|-----------|
| `obsidian_mcp/` | Obsidian 볼트 stdio MCP 서버 — 7개 도구로 LLM 에 볼트 노출 | #51 |
| `doc_agent/` | 이벤트 기반 초안 노트 자동 생성 (백테스트·인시던트·포스트모템) | #53 |

## 경계
- `src/` 의 실행·리스크·체결 로직을 **직접 호출하지 않는다**. 필요하면 이벤트 큐·파일 기반 인터페이스 경유
- LLM 호출은 선택(`--with-llm` 플래그), 기본 동작은 템플릿 기반
- 쓰기 도구는 기본 dry-run, 사람이 승인 후에만 실쓰기
- 자동 커밋·푸시 금지
