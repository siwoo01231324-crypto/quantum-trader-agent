# Incident Root Cause Prompt

다음 인시던트 이벤트에 대해 root cause 초안 1문단을 작성해 달라.

- 발생: {{occurred}}
- 심각도: {{severity}}
- 증상: {{symptom}}
- 시장 맥락: {{market_context}}
- 위반 규칙: {{violated_rules}}
- 영향 전략: {{affected_strategies}}

요구사항:
1. 가설 수준임을 명시 ("가능성이 높다", "추가 분석 필요" 등).
2. 시장 미시구조·실행 알고리즘·리스크 규칙 중 어느 층위 원인인지 분류.
3. 한국어, 3~5문장.
